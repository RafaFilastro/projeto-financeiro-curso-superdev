# TechFix

Projeto desenvolvido em dupla durante a formação Super.Dev.

Sistema para gerenciamento de uma assistência técnica.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- API REST

## Funcionalidades

- Cadastro de clientes;
- Cadastro de serviços;
- Orçamentos;
- Histórico de atendimentos;
- Integração com API.

## Objetivo

Praticar JavaScript e consumo de APIs desenvolvendo um sistema próximo de um cenário real.
